# 教程

如何用 ARC20.ME 显示你的 Realm 资料

:::info

目前，ARC20.ME 还处于早期版本，尚未内置铸造 Realm 和制作 Realm 资料的功能。因此，我们将使用第三方工具来完成领域铸造和资料制作。

:::

接下来，我将详细讲解整个流程，只需按照以下步骤操作，即可完成一个 Realm 资料页面的创建。

## 1. 铸造领域

目前，铸造领域（Realm）主要有三种方法：

- 协议官方的命令行工具（[Atomicals-js Cli](https://github.com/to1dev/atomicals-js)）
- 钱包和交易市场提供的铸造工具（[Wizz 钱包](https://chromewebstore.google.com/detail/wizz-wallet/ghlmndacnhlaekppcllcpcjjjomjkjpg)、[AtomicalMarket 市场](https://atomicalmarket.com/mint/realm)）
- 提供领域铸造功能的第三方网站（[Realmbullrun](https://realmbullrun.com/mint-subrealm)）

:::tip

多数情况下我们会使用方法一和方法二来铸造领域，所以，这里重点讲解这两种方法。其中，第二种方法最为简单，也最适合新手，因此我们先讲解如何使用 Wizz 钱包和 Atomical Market 市场铸造领域。

:::

### 1). 使用 Wizz 钱包铸造

:::warning

这里假定你已经安装好了 Wizz 钱包，并已创建或导入了你的比特币助记词。如果你尚未安装，请先移步至谷歌应用商店安装 [Wizz 钱包](https://chromewebstore.google.com/detail/wizz-wallet/ghlmndacnhlaekppcllcpcjjjomjkjpg)。

:::

下面，让我来演示如何一步步的铸造一个领域（Realm），这里我们将铸造 Bureau 这个名称。

1. 点击谷歌浏览器右上方的 Wizz 钱包图标<img src={wizz_icon} width="32" />，在弹出的主界面上点击 **Mint** 按钮。

<img src={a1} width="400" />

2. 进入钱包自带的铸造页面前，先勾选 **I accept...** 按钮接受条款，然后点击 **I Understand** 按钮。

<img src={a2} width="400" />

3. 在钱包的铸造页面，选择 **Realm** 页，然后选中 **Realm** 选项，在输入框输入你要铸造的 Realm 名称，这里我们输入 bureau。

<img src={a3} width="400" />

4. 如果没有出现错误提示，表示当前名称有效，可以铸造。如果想使用其它地址接收铸造的 Realm，可以勾选 **Received By Other Address**，然后点击 **Mint Realm** 按钮，按照弹出的钱包提示签名，即可完成铸造。

<img src={a4} width="400" />

### 2). 使用 Atomical Market 市场铸造

:::warning

使用 Atomical Market 同样需要安装支持 ARC20 协议的钱包，这里假定你已经安装好了 Wizz 钱包，并已创建或导入了你的比特币助记词。如果你尚未安装，请先移步至谷歌应用商店安装 [Wizz 钱包](https://chromewebstore.google.com/detail/wizz-wallet/ghlmndacnhlaekppcllcpcjjjomjkjpg)。

:::

1. 首先访问 [Atomical Market](https://atomicalmarket.com/mint/realm) 的 Realm 铸造页面，然后点击右上角 **Connect** 按钮。

<img src={am_connect} width="400" />

2. 在弹出的钱包选择框中选择 **Wizz Wallet** 进行连接。（你也可以选择其它钱包，但是 Wizz 钱包对 ARC20 的支持度最高，所以推荐使用 Wizz 钱包）

<img src={a5} width="400" />

3. 钱包连接后，在 Realm 名称输入框中输入你要铸造的名称，这里我们输入 bureau，然后点击 **Next**，在弹出的确认框中点击 **Mint** 按钮。最后，按照弹出的钱包提示进行签名即可完成铸造。

<img src={a6} width="400" />

### 3). 使用协议官方命令行工具铸造

:::tip

要使用协议命令行工具，需要熟练掌握操作系统的控制台，了解如何安装 npm 软件包，以及克隆 GitHub 仓库。这里假定你的系统已经安装了 Git 和 npm 运行环境。（在安装 npm 运行环境后，记得在控制台输入 ```corepack enable``` 激活 **yarn** 命令）

:::

1. 克隆协议命令行源码仓库，安装所需依赖包，然后编译。

```bash showLineNumbers
git clone https://github.com/to1dev/atomicals-js.git
cd atomicals-js
yarn install
yarn build
```

:::info

这里选择克隆 [https://github.com/to1dev](https://github.com/to1dev/atomicals-js) 的协议命令行源码仓库，是由于索引服务器 1.5.1.0 的更新，导致 API 参数有所变化，官方源的命令行在铸造 Realm 时会出现错误，所以暂时选择 [https://github.com/to1dev](https://github.com/to1dev/atomicals-js) 的修正版。一旦我们提交的 PR 通过，将会换回官方源仓库。

:::

2. 确认待铸造的领域名称的有效性。

```bash showLineNumbers
yarn cli get-realm bureau

# 如果得到以下结果，说明这个名称可以铸造
{
  "success": false,
  "data": {
    "atomical_id": null,
    "candidates": [],
    "found_full_realm_name": null,
    "missing_name_parts": "bureau",
    "nearest_parent_realm_atomical_id": null,
    "nearest_parent_realm_name": null,
    "request_full_realm_name": "bureau",
    "top_level_realm_atomical_id": null,
    "top_level_realm_name": null
}
```

3. 铸造领域（这里待铸造的领域名称是 bureau）

```bash showLineNumbers
# **接收地址** 填写你准备用来接收这个领域名称的比特币钱包地址
# **付款地址私钥** 填写你为铸造交易付款的比特币地址的 WIF 私钥（这个私钥也可以提前通过 yarn cli wallet-import 命令导入）
# **交易费** 填写比特币交易矿工费，可以参考 https://mempool.space/ 获取最新值
# **bitworkc** 不懂的直接取默认 8888
yarn cli mint-realm --initialowner=接收地址 --funding=付款地址私钥 --satsbyte=交易费 --bitworkc=8888 bureau
```

## 2. 铸造领域资料

:::tip

目前，领域资料的铸造还是依靠协议命令行工具。如果你还没有安装该工具，请先参考[上面](enable-arc20-me#3-%E4%BD%BF%E7%94%A8%E5%8D%8F%E8%AE%AE%E5%AE%98%E6%96%B9%E5%91%BD%E4%BB%A4%E8%A1%8C%E5%B7%A5%E5%85%B7%E9%93%B8%E9%80%A0)的流程进行安装。

待安装完成后，按照以下步骤制作并铸造领域资料。

:::

### 1). 制作领域资料

领域资料本质上是一个 JSON 文件（JSON 文件是一种用于存储和交换数据的纯文本文件，使用键值对和数组来表示数据结构）。所以，我们可以使用任何文本编辑器编辑领域资料。下面，我们根据协议官方推荐的格式来一步步制作一份领域资料。

✍️ 以下是协议官方提供的一个极简化领域资料示例。

```json showLineNumbers
/*
** 来自 https://github.com/atomicals/atomicals-js/blob/avmbeta1/templates/socialfi/min-profile.json
** 协议官方的领域资料示例，一份领域资料至少应包括以下字段
**
** 版本 v
** 名称 name
** 头像 image
** 简介 desc
**
** 其余字段均为可选，推荐加上 links 字段，用来添加社交媒体和其它网站链接
** 如果希望用户能够给你捐款，还可以加上 wallets 字段，用来添加区块链钱包地址
*/

{
    "v": "1.2",
    "name": "User or profile title or name",
    "image": "atom:btc:id:<atomicalId>/image.jpg",
    "desc": "Lorem ipsum dolor sit amet...",
    "ids": {
        "0": {
            "t": "realm",
            "v": "myrealmname"
        }
    },
    "wallets": {
        "btc": {
            "address": "btc address to receive tips"
        }
    },
    "links": {
        "0": {
            "group": "social",
            "items": {
                "0": {
                    "type": "x",
                    "name": "@loremipsum9872",
                    "url": "https://x.com/loremipsum9872"
                }
            }
        }
    }
}
```

ARC20.ME 还添加了几个额外字段用来显示 banner 和背景图片。这正好体现了 Atomicals 协议的开放性，任何人都可以修改领域资料的格式，添加不同字段以丰富领域的使用场景。

```json showLineNumbers
// 所有图像 URN 既可以是 Atomicals 协议下的引用，也可以是任何有效的图像 URL。
{
    "v": "1.2",
    "name": "User or profile title or name",
    "image": "atom:btc:id:<atomicalId>/image.jpg",
    "banner": "atom:btc:id:<atomicalId>/banner.jpg",
    "background": "atom:btc:id:<atomicalId>/background.jpg",
    "desc": "Lorem ipsum dolor sit amet...",
}
// 或
{
    "v": "1.2",
    "name": "User or profile title or name",
    "image": "https://example.com/image.png",
    "banner": "https://example.com/banner.jpg",
    "background": "https://example.com/background.jpg",
    "desc": "Lorem ipsum dolor sit amet...",
}
```
