# 简介

介绍 ARC20.ME 及其特色

[ARC20.ME](https://arc20.me) 是一个简单而强大的 Atomicals Realm 浏览器，用来展示基于比特币区块链的个人信息页面，你可以将她想象成一个把数据存储在比特币区块链上的 Linktree。允许用户通过 Atomicals 协议创建基于任意 Realm 的结构化数据，并汇总你在不同平台上的所有链接。这些链接可以是社交媒体账户、个人网站、博客文章、在线商店、活动页面等。现在你只需在任何社交媒体简介中放置一个 ARC20.ME 链接，访客点击该链接后就能看到你所有重要链接的汇总页面。

ARC20.ME 还利用比特币区块链技术带来了更多的创新和优势。

1. 去中心化：所有数据存储在比特币区块链上，确保数据的不可篡改和高安全性。
2. 隐私保护：用户数据不会被集中存储或追踪，保障用户隐私。
3. 透明度：所有交易和数据操作都记录在区块链上，任何人都可以验证数据的真实性。
4. 全球可访问：区块链技术保证了网站的全球可访问性和高可用性。
