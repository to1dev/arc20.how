# 简介

## 什么是 Atomicals 数字对象？

Atomicals 数字对象（简称“Atomical”）是一种新型的非同质化代币（NFT），可以在比特币上铸造、转移和更新。其主要区别在于，无需使用中心化服务或充当可信索引器的中间人。它开箱即用，而且不需要对比特币做任何改动，也不需要侧链或任何比特币二层。**是时候永远夺回对我们数字生活的控制权了。**

Atomicals 不言自明且易于验证，这使它们非常适合数字收藏品、社交媒体、游戏、认证以及任何需要确立所有权和真实性的场合。协议规则极其简单，同时提供军事级别的安全性和验证等级。零容错。

Atomicals 协议是免费的、开源的，任何人在任何地方都可以永远随时使用。我们的数字未来依赖于一个健壮的数字对象格式，以优雅地处理每一个可能的用例，同时最小化软件实现中的任何潜在错误。

## 如何创建 Atomicals 数字对象

使用 Taproot (P2TR) 消费脚本、Atomicals 信封以及由字母 "m" 表示的铸造操作，通过一个两步骤的提交和揭示方案来铸造一个 Atomical。交易输出承诺了被转移的数据或文件，然后在消费脚本中包含数据以揭示内容，该内容可以是一个或多个具有任何内容类型的文件，如图片、文本或任何媒体。

使用命令行界面和您现有的比特币钱包，铸造您的第一个 Atomical。

一旦创建，Atomical 将永久不变地存储在区块链上，并且可以使用任何地址类型根据比特币的熟悉规则转移到任何新的所有者手中。
